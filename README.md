# SoilAndAnalysis
This is a coursework for Internet of Everything Lab.

Stack:

 - Cisco Packet Tracker
 - Vue Js
 - Vue Trends
 - Twitter Bootstrap

Steps to run:

    Install Cisco Packet Tracer
    Open PKT file
    Give permissions to external network access from device scripts
    open localhost:8765/index.html
